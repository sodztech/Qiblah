# Project Structure

This root is organized into six top-level project areas:

- `Main Website`
- `Mosque Admin`
- `Super admin`
- `Prayer Tablet`
- `Qiblahiphone`
- `IOS tablet app`

The folders use links to the current live files where those files already existed. This keeps the existing website routes and deployment paths working while making the project easier to browse locally.
